# Weather-Dashboard-App 
### Weather Dashboard 
https://nickmardon.github.io/Weather-Dashboard-App/ \
Built with HTML, CSS, Bootstrap, JavaScript, Jquery, and openweathermap API. 


## User Experience Guide

### 1. Start Page
User may enter a city name into the search bar, and click the search button to view the current weather and a five day forcast for that city.

<img width="1680" alt="Screen Shot 2020-06-24 at 11 31 31 PM" src="https://user-images.githubusercontent.com/64296192/85668100-2b2bb380-b673-11ea-8cd0-cfee86923415.png">

### 2. Weather is Displayed
When user has entered a valid city name and clicked the search button, data will populate the screen about the current weather and five day forcast for the city.

<img width="1680" alt="Screen Shot 2020-06-24 at 11 32 13 PM" src="https://user-images.githubusercontent.com/64296192/85668480-a0978400-b673-11ea-95bd-51c549295ba5.png">

### 3. Additional Inputs
User can continue to add more cities to their searched cities bank.  By default searching a new city will populate it's data on the page.  To change data between previously picked cities, just click their coresponding button in the search bank.
